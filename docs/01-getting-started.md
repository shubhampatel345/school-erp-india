# SHUBH SCHOOL ERP — Getting Started

## First-Time Setup Steps

1. **Configure School Profile** — Settings → School Profile. Enter school name, address, logo. Auto-populates all receipts and certificates.
2. **Set Up Classes & Sections** — Academics → Classes & Sections. Add your class structure (1A, 1B, 2A...12B).
3. **Add Subjects** — Academics → Subjects. Add subjects and assign to multiple classes using the wizard.
4. **Add HR Staff & Teachers** — HR → Staff Directory → Add Staff. For teachers, set subject-class range assignments.
5. **Create Fee Structure** — Fees → Fee Headings: define names with applicable months. Fees → Fee Plan: set section-wise amounts.
6. **Admit Students** — Students → Add Student. Credentials auto-created (Adm.No. / DOB).
7. **Start Collecting Fees** — Fees → Collect Fees. Search student, select months, save and print receipt.
